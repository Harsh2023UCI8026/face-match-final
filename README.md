# 🎭 Bollywood Celebrity Face Matcher

Find out which Bollywood celebrity you look like using AI-powered face recognition.

This project uses **ArcFace (InsightFace)** embeddings + **cosine similarity** to match your face with celebrities.

Built with ❤️ using Streamlit + Computer Vision.

---

## 🚀 Demo

Upload your photo → Get your celebrity twin + similarity score 😄

---

## ✨ Features

✅ Face detection (ArcFace)  
✅ High-accuracy embeddings (512-d)  
✅ Fast matching using cosine similarity  
✅ Upload image support  
✅ Webcam capture (optional)  
✅ Fun Bollywood-style messages  
✅ Download result poster  
✅ Lightweight deployment (no heavy dataset)

---

## 🧠 Tech Stack

- Python 3.10+
- Streamlit
- InsightFace (ArcFace)
- ONNX Runtime
- OpenCV
- NumPy
- scikit-learn
